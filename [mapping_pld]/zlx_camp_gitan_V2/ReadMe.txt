Installation : 
 - Ouvrir le fichier WinRar
 - Sélectionner le fichier zlx_camp_gitan_V2 puis le glisser dans vos ressources
 - Start le Mapping dans votre Server.cfg

- Open WinRar file
- Select the zlx_camp_gitan_V2 file and drag it into your assets
- Start the mapping in your Server.cfg

  - Assurez-vous de passer en mode édition !!

   - Ensuite, prenez le YMAP et glissez-déposez-le dans OpenIV puis c’est dedans!
